# Docker Assignment

## Description
This project demonstrates building and running a Docker container for a simple static web application on an AWS EC2 instance. The project includes:

- Dockerfile for a simple HTML web app
- Running the container and accessing it in a browser
- Optional Docker Compose setup (Web + Database)

## Files Included
- Docker-assignment.docx — Google Docs report
- Dockerfile — Docker instructions for the web app
- index.html — Static HTML page
- docker-compose.yml — Optional multi-container setup
